Investigación de Operaciones

Este es un ejemplo de código para resolver un problema de programación lineal utilizando CVXPY.

Minimizar 3⁢x+8⁢y sujeto a:

  • x+y≥50
  • x-y≤20
  • x≥0
  • y≥0

  import cvxpy as cp

  x = cp.Variable()
  y = cp.Variable()

  constraints = [x + y >= 50, x - y <= 20, x >= 0, y >= 0]
  obj = cp.Minimize(3*x + 8*y)
  prob = cp.Problem(obj, constraints)

  # Resolver utilizando el solver ECOS
  prob.solve(solver=cp.ECOS)
  print("Valor óptimo con ECOS:", prob.value)
  print("Solución óptima con ECOS:")
  print("x =", x.value)
  print("y =", y.value)

  # Resolver utilizando el solver SCS
  prob.solve(solver=cp.SCS)
  print("Valor óptimo con SCS:", prob.value)
  print("Solución óptima con SCS:")
  print("x =", x.value)
  print("y =", y.value)
